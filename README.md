# 🏋️ FitTrack Pro (MERN Workout Tracker)

A full-stack web application built with the **MERN stack** (MongoDB, Express, React, Node.js) that allows users to track their daily workouts, log exercises, and manage their fitness journey.

## ✨ Features
*   **Modern Premium UI:** Features a sleek, responsive Glassmorphism design with dark-mode elements and glowing neon accents.
*   **Full CRUD Operations:** Add new workouts, view all past workouts, and delete entries.
*   **RESTful API:** Robust Express backend connected to a MongoDB database.
*   **Real-time State Management:** React Context API and custom hooks for instant UI updates without refreshing.

## 🚀 Tech Stack
*   **Frontend:** React, Context API, Modern CSS (Glassmorphism)
*   **Backend:** Node.js, Express.js
*   **Database:** MongoDB, Mongoose ORM

## 📦 Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/fit-track-pro.git
    cd fit-track-pro
    ```

2.  **Setup the Backend:**
    ```bash
    cd backend
    npm install
    ```
    *Create a `.env` file in the `backend` folder and add your `MONGO_URI` and `PORT`.*
    ```bash
    npm run dev
    ```

3.  **Setup the Frontend:**
    ```bash
    cd ../frontend
    npm install
    npm start
    ```
